## Proč Claude?

Claude od Anthropic je AI asistent postavený s důrazem na přesnost, bezpečnost a schopnost pracovat s rozsáhlým kontextem. Po roce intenzivního používání ho považuji za nejlepší volbu pro profesionální práci — zejména programování, analýzu dokumentů a strukturované úlohy.

## Modelová řada

**Claude Opus** — nejmocnější model. Exceluje v komplexních reasoning úlohách, analýze kódu a situacích vyžadujících hluboké pochopení kontextu. Pomalejší, dražší, ale výsledky jsou znatelně lepší.

**Claude Sonnet** — nejlepší poměr výkon/cena. Pro 90 % úloh je Sonnet dostačující a výrazně rychlejší než Opus. Doporučuji jako výchozí model.

**Claude Haiku** — nejrychlejší a nejlevnější. Ideální pro jednoduché klasifikace, extrakce dat a real-time aplikace, kde záleží na latenci.

## Extended thinking

Claude Opus a Sonnet umožňují zapnout explicitní reasoning — model před odpovědí "přemýšlí" a ukazuje vám svůj myšlenkový proces. Na rozdíl od OpenAI o1/o3, kde je chain-of-thought skrytý, u Claude vidíte celý postup.

V praxi to zásadně zlepšuje:
- Matematické a logické problémy
- Debugging složitého kódu
- Rozhodování mezi více variantami
- Analýzu kontraktů a právních textů

## Claude Code — killer feature pro vývojáře

Claude Code je CLI nástroj, který mění způsob, jakým programuji. Není to autocomplete jako Copilot — je to agent, který chápe celý projekt:

- **Naviguje repozitář** a rozumí struktuře
- **Píše a edituje soubory** podle vysokoúrovňových instrukcí
- **Spouští testy** a opravuje nalezené chyby
- **Commituje a vytváří PR** s rozumnými commit messages
- **MCP integrace** pro přístup k databázím, API, Figma a dalším nástrojům

Tento web byl kompletně vytvořen přes Claude Code — od scaffoldu přes komponenty po Docker konfiguraci a deploy na Vercel.

## API a integrace

Claude API je straightforward a dobře zdokumentované. Podporuje:

- **Messages API** s system prompty a multi-turn konverzacemi
- **Vision** — analýza obrázků, screenshotů, dokumentů
- **Tool use** — strukturované volání externích funkcí
- **Streaming** pro real-time odpovědi
- **Batches** pro hromadné zpracování

Skvěle funguje s Vercel AI SDK — jeden řádek kódu pro přepnutí mezi Claude, GPT nebo Gemini.
