## VS Code, ale s mozkem

Cursor je fork VS Code s AI integracemi přímo v jádru editoru. Neinstaluješ extension — AI je součást editoru.

### Co ho dělá výjimečným

**Cmd+K (inline edit):** Označíš kód, popíšeš co chceš změnit, a Cursor to upraví přímo v souboru. Žádné copy-paste z chatu. Funguje překvapivě dobře pro malé úpravy.

**Composer:** Multi-file generátor. Popíšeš feature a Composer vytvoří/upraví víc souborů najednou. Pro scaffolding nových komponent je skvělý.

**Chat s kontextem:** AI chat, který automaticky indexuje tvůj projekt. Můžeš se ptát na architekturu, debugging, nebo říct "přidej validaci do tohoto formu" a ví o čem mluvíš.

## Proč dávám "jen" 4 hvězdy

Cursor je výborný editor. Ale mám Claude Code a upřímně — většinu těžké práce dělám v terminálu. Claude Code má lepší kontextové povědomí (čte git historii, spouští testy) a workflow je rychlejší.

Cursor používám na:
- Quick inline edity (Cmd+K je fakt rychlý)
- Vizuální code review (vidím diff přímo v editoru)
- Exploraci neznámého kódu (chat se ptá na projekt)

Claude Code používám na:
- Nové features (scaffolding, multi-file changes)
- Debugging (spustí testy, čte error logy)
- Refactoring (chápe celý projekt)
- Commit a PR workflow

## Cursor vs. VS Code + Copilot

Cursor je lepší než VS Code s Copilotem. Inline edity jsou přirozenější, Composer nemá ekvivalent, a kontextové povědomí je hlubší. Pokud zvažuješ jedno nebo druhé — Cursor.

## Verdikt

Cursor je nejlepší AI editor na trhu. Pokud nepoužíváš Claude Code v terminálu, Cursor je jasná volba. Pokud používáš oboje (jako já), je to silná kombinace — Cursor pro vizuální práci, Claude Code pro těžkou logiku.
