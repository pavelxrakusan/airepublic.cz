## Průkopník, co se nechá předbíhat

Copilot byl první masově úspěšný AI coding tool. Když v roce 2022 vyšel, bylo to zjevení — AI doplňuje kód v reálném čase. Dneska je to baseline, ne výjimka.

## Co dělá dobře

### Autocomplete

Copilot je stále **nejlepší** v inline doplňování. Tab-tab-tab workflow je návykový. Napíšeš název funkce a Copilot doplní tělo. Napíšeš komentář a doplní implementaci. Pro repetitivní kód je to rychlejší než cokoliv jiného.

### GitHub integrace

Copilot zná tvůj repozitář z GitHubu. Navrhuje kód konzistentní s existující codebase, chápe issues a PR kontext. Pro GitHub-centric týmy je to přirozená volba.

## Kde zaostává

### Agentické funkce

Cursor má Composer. Claude Code pracuje s celým repem. Copilot má... autocomplete a chat. Workspace agent (který má pracovat s celým projektem) je v beta a pořád nestabilní.

V roce 2026 jen autocomplete nestačí. Chci nástroj, co chápe celý projekt, generuje multi-file changes a spouští testy. Copilot to zatím neumí na produkční úrovni.

### Chat kvalita

Copilot Chat je slabší než alternativy. Odpovědi jsou často generické, kontext projektu je povrchní. Cursor chat nebo Claude Code jsou výrazně lepší pro otázky typu "proč tohle nefunguje?"

## Pro koho je Copilot v roce 2026?

- Týmy silně navázané na GitHub ekosystém
- Vývojáři co chtějí primárně autocomplete (a nechtějí agenta)
- Firmy s Enterprise licencí (security features, IP protekce)
- Studenti a open source (zdarma!)

## Verdikt

Copilot je dobrý nástroj, ale stagnuje. Za $10/měs je to férová cena za nejlepší autocomplete na trhu. Ale za $20/měs dostaneš Cursor s Composerem nebo Claude Pro s Claude Code — a to je jiná liga. Doporučuji jako doplněk, ne jako primární AI nástroj.
