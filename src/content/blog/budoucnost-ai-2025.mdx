Claude od Anthropic je v mnoha benchmarcích na špičce. Ale co ho skutečně odlišuje od GPT nebo Gemini? Pojďme se podívat pod kapotu.

## Constitutional AI: jiný přístup k alignmentu

Zatímco OpenAI sází primárně na RLHF (Reinforcement Learning from Human Feedback), Anthropic přišel s konceptem **Constitutional AI (CAI)**. Model dostane sadu principů — "ústavu" — a učí se sám sebe hodnotit a opravovat podle těchto pravidel.

V praxi to znamená, že Claude je obecně méně náchylný k jailbreakům a konzistentněji dodržuje instrukce. Na druhou stranu bývá občas příliš opatrný — odmítne i neškodné requesty, pokud je vyhodnotí jako potenciálně problematické.

## Extended thinking: chain-of-thought na steroidech

Claude Opus a Sonnet podporují **extended thinking** — model před finální odpovědí prochází explicitním reasoning procesem. Na rozdíl od OpenAI o1/o3, kde je chain-of-thought skrytý, u Claude si můžete reasoning prohlédnout.

```
// Příklad: API call s extended thinking
const response = await anthropic.messages.create({
  model: "claude-sonnet-4-5-20250514",
  max_tokens: 16000,
  thinking: {
    type: "enabled",
    budget_tokens: 10000
  },
  messages: [{ role: "user", content: "Analyzuj tuto architekturu..." }]
});
```

Extended thinking dramaticky zlepšuje výsledky u:
- Matematických a logických úloh
- Analýzy kódu a debugging
- Složitých rozhodovacích procesů
- Porovnání více variant řešení

## Kontextové okno: 200k tokenů a proč na tom záleží

Claude nabízí 200 000 tokenů kontextu — to je zhruba 500 stran textu. GPT-4 Turbo má 128k, Gemini 1.5 Pro teoreticky až 2M, ale s výrazně klesající kvalitou na dlouhých vstupech.

Klíčový rozdíl není jen velikost, ale **kvalita retrievalu**. V needle-in-a-haystack testech Claude konzistentně nachází informace kdekoliv v kontextu, zatímco jiné modely mají tendenci "zapomínat" prostředek.

Praktické využití:
- Analýza celých codebases (Claude Code načte celý repo)
- Zpracování právních dokumentů nebo smluv
- Code review s plným kontextem projektu
- Sumarizace rozsáhlých research paperů

## Claude Code: AI v terminálu

Jeden z nejzajímavějších produktů Anthropic je **Claude Code** — CLI nástroj, který přímo pracuje s vaším kódem. Na rozdíl od GitHub Copilot, který doplňuje řádky, Claude Code:

- Naviguje repozitář a chápe strukturu projektu
- Píše, edituje a refaktoruje soubory
- Spouští testy a opravuje chyby
- Commituje a vytváří pull requesty
- Pracuje s MCP servery pro přístup k externím nástrojům

Tento web je živý důkaz — byl kompletně vygenerován přes Claude Code v jedné session.

## Kdy použít co?

- **Claude** — programování, analýza dokumentů, přesné instrukce, code review
- **GPT-4o/o3** — kreativní psaní, multimodální úlohy, ekosystém pluginů
- **Gemini** — práce s velkými daty, integrace s Google ekosystémem, cena

Žádný model není univerzálně nejlepší. Klíč je znát silné stránky každého a používat správný nástroj pro daný úkol.
