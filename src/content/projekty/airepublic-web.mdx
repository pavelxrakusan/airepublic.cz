## O projektu

airepublic.cz je český informační portál o umělé inteligenci. Celý web byl vytvořen metodou vibe coding — tedy spolupráci s AI asistentem Claude Code, který generoval většinu kódu na základě mých instrukcí.

## Tech stack

- **Next.js 15** s App Routerem pro server-side rendering a statické generování
- **TypeScript** pro typovou bezpečnost
- **Tailwind CSS v4** pro styling s CSS-first konfigurací
- **MDX** pro obsah — články, recenze a projekty jsou psané v Markdownu s frontmatter metadaty
- **Docker** pro konzistentní vývojové prostředí
- **Vercel** pro hosting a deployment

## Jak vznikal

1. Definoval jsem strukturu webu a požadavky v jednom podrobném promptu
2. Claude Code vygeneroval celou strukturu projektu, komponenty, stránky i obsah
3. Iteroval jsem nad designem a funkčností přes konverzaci
4. Docker zajistil konzistentní běhové prostředí
5. Deploy na Vercel proběhl automaticky po push do repozitáře

## Co jsem se naučil

Vibe coding je překvapivě efektivní pro projekty typu informační web nebo blog. Klíčem je dobře definovaný prompt s jasnou strukturou, tech stackem a designovými požadavky. AI zvládne vygenerovat funkční základ, který pak stačí doladit.
